## 👀 프로젝트 빌드 및 실행 방법

1. 상단 `Code` 버튼을 눌러 레포지토리를 클론 받습니다.

```
$ git clone https://github.com/Jeong-jeong/exchangeCalculator.git
```

2. 패키지를 설치합니다.

```
$ yarn install
```

3. 로컬에서 실행합니다.
```
$ yarn start
```
